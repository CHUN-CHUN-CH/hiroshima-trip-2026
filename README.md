# 2026 廣島朋友旅行

這是一個可直接公開部署的靜態行程網站，包含：

- 2026/7/23-7/29 廣島每日行程
- 嚴島神社必集合行程
- 廣島港花火大會安排
- 多人旅行自由活動與分組日
- 點到點交通時間

## 本機預覽

直接打開 `index.html` 即可。

## 上傳到 GitHub

```powershell
git add .
git commit -m "Add Hiroshima trip itinerary site"
git remote add origin https://github.com/YOUR_ACCOUNT/YOUR_REPO.git
git push -u origin master
```

如果要開 GitHub Pages，到 repo 的 Settings > Pages，選擇從 `master` 分支的 root 目錄發布。
